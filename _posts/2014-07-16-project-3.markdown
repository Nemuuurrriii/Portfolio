description: >
  Developed an internal workflow system used by 500–1,000 active users to manage employee requests such as leave applications, overtime tracking, and approval processes.

  Built and maintained backend APIs using Spring Boot, implementing secure authentication and role-based access control with Spring Security and JWT. Improved system performance through database query optimization.

  Unlike dynamic workflow systems, this implementation focuses on predefined forms with configurable approval routing, allowing organizations to customize approval flows. Implemented overtime listing functionality where employees can apply for available overtime schedules.

  Additional features include notifications, access logging, and audit tracking. Contributed to both frontend and backend development within a collaborative Agile team.
features:
  - Predefined request forms
  - Configurable approval routing
  - Overtime listing and application system
  - Role-based access control (RBAC)
  - Notifications system
  - Access logs and audit tracking
